###  概要

これは、能登地方に伝わる伝統的な対戦型テーブルゲーム**「ごいた」**をWebブラウザで手軽に遊べるように実装したものです。

このバージョンはCPU対戦を楽しむことができます。

###  ゲームをプレイする
